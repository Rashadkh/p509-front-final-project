Client Line-Up

Bank Tabungan Negara

Project WB-Bank BTN : Application Development Project Teller System or known as Web Applications Branch for Bank BTN. Project is to replace the existing application at Bank BTN is running under the Windows platform or desktop applications into Web-based applications applications.

Branch Web application will be implemented throughout the Bank BTN - Indonesia, about 600 BTN Office.

Web framework: tapestry 5.3
Back end : spring 3.1 + jpa 2
Security: apache shiro
Database: ibm db2 9.7
